# To-Do List (DOM)

## Live Demo

[View Project](https://nathan-ribeiro-de-souza.github.io/todo-list-dom-js/)

An interactive to-do list application built with HTML, CSS, and JavaScript, focused on DOM manipulation and dynamic user interaction.

---

## About the Project

This project is a task management application that allows users to add and remove tasks in real time. It was developed to strengthen core front-end skills, especially JavaScript DOM manipulation and event handling.

The goal was to simulate a simple but practical real-world feature while maintaining clean and organized code.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## Features

* Add new tasks dynamically
* Remove tasks instantly
* Real-time DOM updates
* Light/Dark mode toggle
* Organized and responsive layout

---

## What I Learned

* Selecting and manipulating DOM elements
* Handling user events (click, input, keyboard)
* Updating the UI dynamically
* Managing application state on the front-end
* Structuring small interactive applications
